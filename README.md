# GallowsBot
Telegram bot with which you can play the popular game "Gallows"
## Features
##### 1. Different difficulty levels
##### 2. Player stats
##### 3. Game analysis
##### 4. Possibility to add your words locally
## Dependencies 
##### 1. aiogram
##### 2. sqlite3
##### 3. json
##### 4. random
## Codestyle linters and test frameworks

##### 2. mypy
##### 3. pydocstyle
